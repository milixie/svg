### 入门

#### 基本要素

- `<svg>`根元素，相当于HTML里面的标签

- `<g>`元素，用来把若干个基本形状组成一个组

** 注意 **	

- SVG 属于XML语言，因为XML是区分大小写的，所以SVG的属性和元素必须按照标准规范书写

- SVG 的属性值必须使用引号引起来
